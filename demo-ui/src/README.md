# Angular Node Intro

En este respositorio se encuentran dos proyectos:
 - demo-api: proyecto node
 - demo-ui: proyecto angular
 
 ## Instalacion

 - demo-api
   - Ir a la carperta demo-api
   - Abrir un terminal
   - `npm i`
   - `node .`
 - demo-ui
   - Ir a la carpeta demo-ui
   - Abrir un terminal
   - npm i
   - `ng serve`
